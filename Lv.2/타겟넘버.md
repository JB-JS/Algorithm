# 오픈채팅방

## 풀이과정

1. 재귀함수와 dfs 알고리즘을 통해서 주어진 숫자들의 모든 (+, -)가 target 넘버와 같아지는 경우의수를 반환한다.

```javascript
function solution(numbers, target) {
  let answer = 0;

  function dfs(i, num) {
    if (num === target && i === numbers.length) {
      answer += 1;
      return;
    }

    if (i < numbers.length) {
      dfs(i + 1, num + numbers[i]);
      dfs(i + 1, num - numbers[i]);
    }
  }

  dfs(0, 0);

  return answer;
}
```
